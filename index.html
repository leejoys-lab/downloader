릴스 커버 아카이브

공개 릴스 링크를 붙여넣으면 표지 이미지와 영상을 분리해 저장합니다. 브라우저가 Instagram을 직접 호출하지 않고 같은 사이트의 API가 미디어 주소를 확인하므로, Vercel에 프로젝트 전체를 배포해야 작동합니다.
01 · 릴스 링크 붙여넣기 (한 줄에 하나씩, 여러 개 가능)
0개 링크 인식됨
⚠ 이 버전은 공개 CORS 프록시를 사용하지 않습니다. 링크 해석은 /api/resolve, 파일 전송은 /api/media에서 처리합니다. Instagram의 접근 정책이나 요청 제한에 따라 일부 공개 릴스도 실패할 수 있습니다.
💡 실행 전 확인
이 파일을 컴퓨터에서 더블클릭해 file://로 열면 API가 없어서 작동하지 않습니다. 아래 제공한 index.html · api 폴더 · package.json을 함께 GitHub에 올리고 Vercel에서 배포하세요.

본인 계정 전체 백업이 목적이라면 Instagram의 공식 정보 다운로드 기능이 가장 안정적입니다.
